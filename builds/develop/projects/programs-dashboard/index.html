The solution

A centralized repository of icons hosted on Github.com. Moreover, the repository is public facing, meaning our distributors, partners, and customers can leverage Esri's custom-built iconography. Therefore, Esri will set the standard of GIS and mapping icons, which can be adopted by everyone. Additionally, the repository can be sourced vis NPM, so that whenever there is a new release of icons users will be notified of updates, deletion or additions to the library. This will ensure that everyone leveraging this icons system is always up to date with the latest. Creating consistency across the platform.

  • Icons come in two styles: outline and filled (bold)
  • Icons available in 3 sizes (16, 24, and 32)
  • All icons come with search tags to ensure users are using the appropriate concept
  • There are also SVG sprites with all icons
  • `Spriter` CLI took available for building sprites of only the icons needed
  • Over 400 icon concepts
